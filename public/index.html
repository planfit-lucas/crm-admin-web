캠페인 생성 체크리스트

실수를 방지하기 위해 캠페인 생성 전 확인하세요

✅ 내부 테스트 발송 여부 체크 해제 전, 반드시 테스트 먼저!
✅ A/B 테스트 시 변형 트래픽 합계가 100%인지 확인
✅ 발송 대상 SQL 쿼리 결과를 JSON 프리뷰에서 검증
✅ 한국 / 미국 콘텐츠 모두 빠짐없이 입력했는지 확인
✅ 이메일 채널: 발송 이메일(From)이 선택되었는지 확인
✅ 이벤트 기반: Amplitude 이벤트가 올바르게 연결되었는지 확인
✅ 정기 발송: 반복 주기와 시간이 의도와 일치하는지 확인
⚠️ 내부 테스트 발송이 해제되면 실제 유저에게 발송됩니다 — 항상 주의하세요!
